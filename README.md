# PulseCart AI

PulseCart AI is a fully responsive, modern storefront building the foundation for AI-powered commerce. It delivers a fast, dynamic user interface with a robust, localized product catalog.

## ✨ Features

- **Dynamic Product Catalog**: Features a range of categories including Headphones, Smartwatches, Laptops, Cameras, and Keyboards.
- **Shopping Cart System**: Integrated interactive cart drawer with real-time quantity controls, subtotal summaries, and delivery fee thresholds.
- **Wishlist Management**: Dedicated wishlist drawer to save favorite items, with quick "Move to Cart" capabilities.
- **Instant Search & Filters**: Client-side filtering via search keywords or category tabs.
- **Component UI**: Includes high-quality visual elements like image placeholders, dynamic product badges (e.g., "Best Seller", "Pro Pick"), and notification toasts.

## 🛠️ Technology Stack

- **Framework**: Next.js App Router (`app/`)
- **UI/Components**: React Client Components
- **Styling**: Vanilla CSS (`globals.css`) emphasizing smooth micro-animations, glassmorphism, and responsive grid layouts.
- **Icons**: Custom SVG React wrappers (`components/Icons.js`).
- **Data**: Static JSON datastore (`data/products.json`) engineered for future RAG AI integrations.

## 🚀 Getting Started

First, install the dependencies:
\`\`\`bash
npm install
\`\`\`

Run the development server:
\`\`\`bash
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
